# Korea Coding Vibe Lab 🇰🇷💻✨

코딩 초보자를 위한 "바이브(Vibe)" 중심 커뮤니티, **Korea Coding Vibe Lab**에 오신 것을 환영합니다.

## 🚀 시작하는 방법 (How to Start)

터미널에서 다음 명령어를 순서대로 실행해주세요:

1. **의존성 설치 (Install Dependencies)**
   ```bash
   npm install
   ```
   *(이 과정은 인터넷 환경에 따라 시간이 조금 걸릴 수 있습니다.)*

2. **개발 서버 실행 (Start Server)**
   ```bash
   npm run dev
   ```

3. **브라우저 확인**
   터미널에 표시된 `Local: http://localhost:5173` 주소를 브라우저(Chrome 권장)에서 열어주세요.

## ✨ 주요 기능 (Features)

- **🏠 홈 (Dashboard)**: 오늘의 코딩 바이브를 확인하고 동기부여를 얻으세요.
- **📅 출석 체크 (Attendance)**: 매일 출석해서 연속 출석일(Streak)을 쌓아보세요.
- **💬 커뮤니티 (Lounge)**: 익명으로 자유롭게 코딩 이야기를 나누세요.
- **ℹ️ 바이브 소개 (About)**: '즐기는 코딩'이 무엇인지 알아보세요.
- **🤝 바이브 메이트 (Study)**: 함께 성장할 동료를 찾는 공간 (준비 중).

## 🛠️ 기술 스택 (Tech Stack)

- **Framework**: React + Vite
- **Styling**: Vanilla CSS (Glassmorphism & Dark Theme)
- **Icons**: Lucide React
- **Routing**: React Router DOM
